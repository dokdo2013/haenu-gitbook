# i.M 택시 백오피스 개발

* 기간 : 2021년 10월 1일 \~ 2022년 4월 15일
* 회사 : 엠에이치큐 (아이엠택시)

### Backend & Infrastructure

* PHP CodeIgniter 4 (PHP 7.2)

* Linux, Tencent Cloud

### Etc.
